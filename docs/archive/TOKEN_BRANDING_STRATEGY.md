# Token Branding Strategy - Name, Ticker, and Visual Identity

## Executive Summary

Strategic recommendations for token naming and branding that balance **professional credibility** with **crypto-native appeal** while aligning with the platform's **cyberpunk/tech aesthetic**. The goal is a brand that scales from institutional adoption to community meme culture.

**Recommended Primary**: **PROPHECY (PRPH)** - Mystical foresight meets tech sophistication.

## Brand Guidelines (To Develop Pre-Launch)

### Logo Usage
- **Minimum Size**: 32px (digital), 0.5in (print)
- **Clear Space**: Minimum padding = logo height × 0.5
- **Backgrounds**: Works on dark (primary) and light (secondary)
- **Color Variations**: Full color, monochrome, white, black

### Typography
**Primary Font**: Inter
- Headings: Inter Bold (700)
- Body: Inter Regular (400)
- Captions: Inter Medium (500)

**Display Font**: Orbitron (for "PROPHECY" wordmark)
- Used sparingly for brand name only
- Not for body text or UI elements

**Monospace**: JetBrains Mono
- Code snippets
- Data displays
- Technical documentation

### Color System
**Primary Palette**:
```css
--color-primary: #00D4FF;      /* Electric Blue */
--color-secondary: #6B4CE6;    /* Deep Purple */
--color-accent: #00FFF0;       /* Neon Cyan */
--color-background: #0A0E27;   /* Dark Navy */
--color-text: #FFFFFF;         /* White */
```

**Semantic Colors**:
```css
--color-success: #00FF88;      /* Neon Green */
--color-warning: #FFD600;      /* Bright Yellow */
--color-error: #FF006E;        /* Hot Pink */
--color-info: #00D4FF;         /* Electric Blue */
```

**Gradients**:
```css
--gradient-primary: linear-gradient(135deg, #6B4CE6 0%, #00D4FF 100%);
--gradient-secondary: linear-gradient(135deg, #9D4EDD 0%, #00FFF0 100%);
--gradient-accent: linear-gradient(135deg, #FF006E 0%, #FFD600 100%);
```

### Voice & Tone
**Professional Contexts** (docs, whitepapers, investor materials):
- Authoritative
- Technical
- Precise
- Forward-thinking

**Community Contexts** (Twitter, Discord, blog):
- Confident
- Knowledgeable
- Approachable
- Slightly playful

**Marketing Contexts** (landing page, ads):
- Bold
- Aspirational
- Clear value prop
- Action-oriented
