Oracle Bro is the memetic embodiment of prediction market wisdom—part prophet, part degenerate, fully self-aware. ### Code Example: Auto-Reply Bot

```typescript
// oracle-bro-bot/src/index.ts
import { TwitterApi } from 'twitter-api-v2';
import OpenAI from 'openai';
import { getTrendingMarkets, getMarketOdds } from './movemarket-api';

const twitter = new TwitterApi({
  appKey: process.env.TWITTER_API_KEY,
  appSecret: process.env.TWITTER_API_SECRET,
  accessToken: process.env.TWITTER_ACCESS_TOKEN,
  accessSecret: process.env.TWITTER_ACCESS_SECRET,
});

const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });

async function handleMention(tweet: Tweet) {
  const trendingMarkets = await getTrendingMarkets(3);
  const marketData = trendingMarkets.map(m =>
    `${m.question}: ${m.odds}% | $${m.volume}`
  ).join('\n');

  const systemPrompt = `You are @OracleBro... [full prompt from above]

CURRENT CONTEXT:
${marketData}`;

  const response = await openai.chat.completions.create({
    model: 'gpt-4',
    messages: [
      { role: 'system', content: systemPrompt },
      { role: 'user', content: tweet.text }
    ],
    max_tokens: 100,
    temperature: 0.8,
  });

  const reply = response.choices[0].message.content;

  await twitter.v2.reply(reply, tweet.id);

  console.log(`Replied to @${tweet.author_username}: ${reply}`);
}

// Listen for mentions
const stream = await twitter.v2.searchStream({
  'tweet.fields': ['author_id', 'created_at'],
  expansions: ['author_id']
});

stream.on('data', async (tweet) => {
  if (tweet.data.text.includes('@OracleBro')) {
    await handleMention(tweet.data);
  }
});

stream.on('error', (error) => {
  console.error('Stream error:', error);
});
```

3. **Be helpful**: Link to markets, explain odds, educate

3. **Don't spam**: Max 15 tweets/day, space them out
4. **Don't argue**: If someone disagrees, redirect to market odds

7. **Don't over-explain**: Keep it punchy and memetic
