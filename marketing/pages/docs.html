Quick Start

Everything you need to run Move Market locally or integrate it with your own product—on both Aptos and Sui.

Repository Structure

The monorepo ships with smart contracts (contracts/ for Aptos, contracts-sui/ for Sui), backend services (backend/), the React dApp (dapp/), and extensive documentation (docs/). 

git clone https://github.com/movemarket/move-market.git
cd move-market

# Install dependencies
cd dapp && npm install
cd ../backend && npm install

# Start local environment (multi-chain ready)
npm run dev --prefix backend
npm run dev --prefix dapp
