Quick Start

Everything you need to run Based locally or integrate it with your own product.

Repository Structure

The monorepo ships with Solidity contracts (contracts-base/), backend services (backend/), the React dApp (dapp/), and extensive documentation (docs/). 

git clone https://github.com/Archon-444/Based.git
cd Based

# Install dependencies
cd dapp && npm install
cd ../backend && npm install

# Start local environment
npm run dev --prefix backend
npm run dev --prefix dapp
